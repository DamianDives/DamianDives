### 📂 Featured Projects

#### 🌊 [Divesoft-Parser](https://github.com/DamianDives/divesoft-parser) 
A specialized Python utility for parsing **Divesoft DLF** files. Designed to make dive data accessible for researchers and tech-divers who want to dive deeper into their logs.

#### 📊 [Ratio-Dumper](https://github.com/DamianDives/ratio-dumper) 
A dedicated tool for dumping and processing data from the **Ratio iX5M** dive computer, ensuring your underwater stats are never locked behind proprietary software.

#### 📄 [Pikepdf-Annots](https://github.com/DamianDives/pikepdf-annots)
A collection of helper utilities for **pikepdf** that simplifies the management of PDF annotations. Great for automating document workflows and metadata handling.
